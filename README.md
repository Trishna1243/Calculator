# Calculator App

A simple calculator made with HTML, CSS, and JavaScript.

## How to Use
- Open `index.html` in your browser.
- Click buttons to do calculations.

## How to Install
1. Download or clone this repo.
2. Open the project folder.
3. Double-click `index.html`.
